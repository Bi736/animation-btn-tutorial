HOW TO MAKE A ANIMATION BUTTON

You should have clear knowledge about position absolute at this first stage.

Stage: One

Output result

  • Top: 0;( It stands on the top )

  • Left: -100%;( It stands on the left )

  • 100%{Left: 100%}( It will travell in reverse. )

Stage: two

Output result

  • Top: -100%;( It stands on the top )

  • Right: 0%;( It stands on the right )

  • 100%{Right: 100%}( It will travell in reverse. )

Stage: Three

Output result

  • Bottom: 0;( It stands on the down )

  • Right: -100%;( It stands on the right )

  • 100%{Left: 100%}( It will travell in reverse. )

Stage: Four

Output result

  • Left: 0;( It stands on the left )

  • Bottom: -100%;( It stands on the bottom )

  • 100%{Bottom: 100%}( It will travell in reverse. )
